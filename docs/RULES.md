# 斗地主规则（本项目单一真相）

## 牌面
3~A, 2, 小王(BJ), 大王(RJ)

## 牌型（每种 >=2 正例 + >=1 反例）
- 单牌：`[3]`, `[A]`; 反例 `[3,4]`
- 对子：`[7,7]`, `[K,K]`; 反例 `[7,8]`
- 三张：`[9,9,9]`, `[A,A,A]`; 反例 `[9,9,10]`
- 三带一：`[6,6,6,8]`, `[Q,Q,Q,3]`; 反例 `[6,6,7,8]`
- 三带二：`[6,6,6,8,8]`, `[J,J,J,4,4]`; 反例 `[6,6,6,8,9]`
- 顺子：`[3,4,5,6,7]`, `[7,8,9,10,J,Q]`; 反例 `[10,J,Q,K,A,2]`
- 连对：`[3,3,4,4,5,5]`, `[8,8,9,9,10,10,J,J]`; 反例 `[3,3,4,4,6,6]`
- 飞机：`[3,3,3,4,4,4]`, `[7,7,7,8,8,8,9,9,9]`; 反例 `[3,3,3,5,5,5]`
- 飞机带单：`[3,3,3,4,4,4,7,8]`, `[8,8,8,9,9,9,3,4]`; 反例 `[3,3,3,4,4,4,7,7]`
- 飞机带对：`[3,3,3,4,4,4,7,7,8,8]`, `[5,5,5,6,6,6,9,9,10,10]`; 反例 `[3,3,3,4,4,4,7,8,8,8]`
- 四带二（本项目采用：四带两张单牌）：`[9,9,9,9,4,5]`, `[A,A,A,A,3,J]`; 反例 `[9,9,9,9,4,4]`
- 炸弹：`[7,7,7,7]`, `[2,2,2,2]`; 反例 `[7,7,7,8]`
- 火箭：`[BJ,RJ]`, `[RJ,BJ]`; 反例 `[BJ,BJ]`

## 比较规则
- 同牌型比较 `mainRank`（及长度一致）
- 炸弹 > 非炸弹
- 火箭最大

## 跟牌规则
- 必须同牌型且更大，否则 PASS
- 例外：炸弹/火箭可打断普通牌型
